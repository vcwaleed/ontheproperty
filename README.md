# ONTHEPROPERTY

ONTHEPROPERTY is a web application built with Next.js and Tailwind CSS for the frontend, and Express with MongoDB for the backend. This project aims to provide users with a seamless platform for property listings and real estate management.

## Project Repository

[GitHub Repository - ONTHEPROPERTY](https://github.com/vcwaleed/ontheproperty/tree/dev1)

---

## Tech Stack

### Frontend:

- **Next.js**: React framework for server-side rendering and static site generation.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.

### Backend:

- **Express**: Minimalist web framework for Node.js.
- **MongoDB**: NoSQL database for data storage.

---

## Features

- Property listings with detailed descriptions.
- User authentication and authorization.
- Responsive and mobile-friendly design.
- Backend API built with Express.
- Database management using MongoDB.

## Installation and Setup

### Prerequisites

- Node.js and npm installed
- MongoDB installed and running

### Steps to Run Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/vcwaleed/ontheproperty.git
   cd ontheproperty
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**

   - Create a `.env` file in the root directory.
   - Add necessary environment variables (e.g., MongoDB URI, JWT secret).

4. **Run the backend server:**

   ```bash
   npm run server
   ```

5. **Run the frontend:**

   ```bash
   npm run dev
   ```

6. **Access the application:**

   - Visit `http://localhost:3000` in your browser.
